# ✅ GETTING STARTED CHECKLIST

Print this or keep it open. Check boxes as you go!

## ⚡ Quick Setup (30 minutes)

### Step 1: Prerequisites (5 min)
- [ ] Node.js 18+ installed (check: `node --version`)
- [ ] Text editor ready (VS Code recommended)
- [ ] Terminal/Command Prompt open

### Step 2: Accounts Setup (10 min)
- [ ] Supabase account created (supabase.com)
- [ ] Groq account created (console.groq.com)
- [ ] GitHub account (github.com) - for deployment later

### Step 3: Project Setup (5 min)
- [ ] Downloaded/extracted this folder
- [ ] Opened folder in terminal
- [ ] Ran `npm install` (wait 2-3 min)

### Step 4: Database Setup (5 min)
- [ ] Created Supabase project
- [ ] Copied supabase-schema.sql content
- [ ] Pasted in Supabase SQL Editor
- [ ] Clicked "Run" - saw "Success"
- [ ] Got Supabase URL and keys from Settings → API

### Step 5: API Keys (3 min)
- [ ] Got Groq API key from console.groq.com
- [ ] Copied .env.example to .env
- [ ] Filled in all values in .env file
- [ ] Generated NEXTAUTH_SECRET (command in QUICKSTART.md)

### Step 6: First Run (2 min)
- [ ] Ran `npm run dev`
- [ ] Opened http://localhost:3000
- [ ] Saw the homepage load
- [ ] Celebrated! 🎉

## 🚨 Red Flags (Stop & Fix Immediately)

⚠️ **If you see:**
- [ ] Zero signups after 1 week → Landing page problem
- [ ] Signups but zero usage → Onboarding problem
- [ ] High usage but no conversions → Pricing/value problem
- [ ] Negative feedback → Core product problem

**Action:**
1. Stop building new features
2. Talk to 10 users
3. Find the real issue
4. Fix it before proceeding
